What?

官方文档：

大意如下：mongo的聚合从多个文档分组进行处理，最终返回一个结果。并且包括三种：

  1. pipeline
  2. MapReduce
  3. 单一目的聚合操作

Pipeline

pipeline就是按照用户定义的stage，一个阶段一个阶段的执行，上一个阶段的输出作为下一个阶段的输入。
